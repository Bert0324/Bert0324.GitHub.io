常用Linux命令

top

查看系统运行情况

  • top

ps

查看系统进程情况

  • ps -ef

netstat

查看本机端口占用

  • netstat -tunlp

lsof

查看特定端口占用

  • lsof -i:PORT

tail

显示最新追加内容

  • tail -f FILE_PATH

指定行数

  • tail -n NUMBER FILE_PATH

dig

追踪dns解析过程

  • dig @DNS_URL +trace URL

free

查看当前内存占用情况(unit: mb)

  • free -m

kill

通过pid杀死进程

  • kill -QUIT PID

scp

mv & cp

git

设置remote地址

  • git remote add origin URL

重新设置remote地址

  • git remote set-url origin URL

merge all incoming

  • git pull -X theirs
